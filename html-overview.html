مقدمة في الHTML

HTML هي اللغة الأساسية لتكوين صفحات الويب. تستخدم لوصف بنية صفحات الويب باستخدام علامات (tags). كل عنصر HTML يكون مغلفًا بعلامة بداية وعلامة نهاية.

أمثلة على العناصر الأساسية في HTML:


<!DOCTYPE html>
<html>
  <head>
    <title>صفحة مثال</title>
  </head>
  <body>
    <h1>مرحبًا بكم في موقعي</h1>
    <p>هذه فقرة نصية.</p>
  </body>
</html>

ملخص العلامات في HTML

اسم العلامة مثال للاستخدام الهدف الخصائص التي يجب إضافتها إلى العلامة (Attributes) العرض: مضمن (inline) أو كفقرة منفصلة (block) نوع الإغلاق: محكم أو داخلي ميزات التصميم (التشديد، مائل، تسطير، يتوسطه خط، حجم الخط)
h1-h6 <h1>Headline</h1> عناوين - block محكم حجم الخط حسب اسم العلامة (مثل h1 أكبر حجم)
p <p>a long paragraph</p> فقرة - block محكم -
img <img src="www" alt="picture"> صورة src, alt inline داخلي -
a <a href="url">Link</a> رابط href inline محكم يتغير لون النص والتسطير وفقًا للحالة (الأزرق قبل النقر، والأرجواني بعد النقر)
ol <ol><li>Coffee</li><li>Tea</li><li>Milk</li></ol> قائمة مرقمة - block محكم النزول سطر قبل وبعد
ul <ul><li>Coffee</li><li>Tea</li><li>Milk</li></ul> قائمة غير مرقمة - block محكم النزول سطر قبل وبعد
li <li>Coffee</li> عنصر في القائمة - block محكم إضافة رمز نقطي/رقم قبل السطر
strong <strong>Important text</strong> تشديد / تظليل - inline محكم تظليل / تشديد
em <em>had</em> مائل / خط مائل - inline محكم مائل / خط مائل
