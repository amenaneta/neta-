display:

  • block: يجعل العنصر يأخذ عرض الصفحة بالكامل ويبدأ في سطر جديد. القوائم العمودية عادةً تستخدم هذا.
  • inline: يجعل العنصر يظهر في نفس السطر دون كسر السطر. يمكن استخدامه لعناصر القائمة الأفقية.
  • inline-block: يجمع بين خصائص block و inline، حيث يعرض العنصر في نفس السطر لكنه يسمح بتحديد عرض وارتفاع.
  • none: يخفي العنصر من الصفحة، مما يجعله غير مرئي.

position:

  • static: الوضع الافتراضي، لا يمكن التحكم بموقع العنصر باستخدام خصائص top، right، bottom، أو left.
  • relative: يمكن تحريك العنصر من موقعه الطبيعي باستخدام الخصائص المذكورة أعلاه.
  • absolute: يجعل العنصر يتموضع بالنسبة إلى أقرب عنصر أبوي ذو خاصية position غير static.
  • fixed: يجعل العنصر ثابتًا بالنسبة إلى نافذة العرض حتى مع التمرير.
  • sticky: مزيج من relative و fixed، حيث يبقى العنصر "ثابتًا" عند تمرير الصفحة.

list-style:

  • none: لإزالة النقط أو الأرقام من عناصر القائمة.
  • disc, circle, square: لتغيير نوع النقط في القوائم غير المرتبة.
  • decimal, lower-alpha, upper-alpha: لتغيير نوع الأرقام أو الحروف في القوائم المرتبة.

float:

  • left, right: تجعل العنصر يطفو إلى اليسار أو اليمين، وغالبًا ما تستخدم لإنشاء القوائم الأفقية.
  • none: الوضع الافتراضي حيث لا يطفو العنصر.

flex (عند استخدام display: flex):

  • flex-direction: يحدد اتجاه عناصر الفلكس (مثل row للقوائم الأفقية و column للقوائم العمودية).
  • justify-content: لضبط توزيع المساحة بين العناصر (مثل space-between، space-around، center).
  • align-items: لضبط المحاذاة العمودية للعناصر (مثل flex-start، flex-end، center).

margin و padding:

  • margin: لضبط المسافة الخارجية حول العنصر.
  • padding: لضبط المسافة الداخلية داخل العنصر.

تطبيق الخصائص على القوائم:

  • القائمة الأفقية: يمكن استخدام display: inline أو display: inline-block على عناصر القائمة، أو display: flex مع flex-direction: row.
  • القائمة العمودية: يمكن استخدام display: block أو display: flex مع flex-direction: column.
  • القائمة المنسدلة: تتطلب استخدام خصائص position: absolute لعناصر القائمة الفرعية لجعلها تظهر تحت العنصر الأب عند التفاعل (مثل التحويم فوق العنصر).
  • القائمة أسفل الصفحة: يمكن استخدام position: fixed مع bottom: 0 لضمان بقاء القائمة في أسفل الصفحة.
  • قائمة breadcrumb: غالبًا ما تستخدم display: inline أو display: inline-block لعناصر القائمة مع بعض التباعد باستخدام margin أو padding.

القوائم الأفقية


    .horizontal-list {
        display: flex;
        flex-direction: row;
        list-style: none;
        padding: 0;
    }
    .horizontal-list li {
        margin-right: 10px;
    }

القوائم العمودية


    .vertical-list {
        display: block;
        list-style: disc inside;
        padding: 0;
    }
    .vertical-list li {
        margin-bottom: 10px;
    }

القوائم المنسدلة


    .dropdown {
        position: relative;
        display: inline-block;
    }
    .dropdown-content {
        display: none;
        position: absolute;
        background-color: #f9f9f9;
        min-width: 160px;
        box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
        z-index: 1;
    }
    .dropdown:hover .dropdown-content {
        display: block;
    }
    .dropdown-content a {
        color: black;
        padding: 12px 16px;
        text-decoration: none;
        display: block;
    }
    .dropdown-content a:hover {
        background-color: #f1f1f1;
    }

القائمة أسفل الصفحة


    .footer-list {
        position: fixed;
        bottom: 0;
        width: 100%;
        background-color: #333;
        color: white;
        text-align: center;
        list-style: none;
        padding: 10px 0;
        margin: 0;
    }
    .footer-list li {
        display: inline;
        margin: 0 15px;
    }

قائمة breadcrumb


    .breadcrumb {
        display: flex;
        list-style: none;
        padding: 8px 0;
        background-color: #f5f5f5;
    }
    .breadcrumb li {
        margin-right: 10px;
    }
    .breadcrumb li::after {
        content: "/";
        margin-left: 10px;
    }
    .breadcrumb li:last-child::after {
        content: "";
    }
