مقدمة في CSS

CSS هي لغة لتنسيق صفحات الويب. تستخدم للتحكم في مظهر عناصر HTML، مثل الألوان، الخطوط، الأحجام، والتخطيط.

أمثلة على استخدام CSS:


<html>
  <head>
    <style>
      body {
        background-color: lightblue;
      }
      h1 {
        color: navy;
        margin-left: 20px;
      }
    </style>
  </head>
  <body>
    <h1>هذا عنوان</h1>
    <p>هذه فقرة.</p>
  </body>
</html>
